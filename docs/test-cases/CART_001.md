# CART_001 — Добавление нескольких товаров в корзину

## Приоритет
Critical

## Функциональность
Корзина

## Предусловия
- Приложение SauceDemo доступно.
- Пользователь авторизован.
- Открыта страница Products.

## Тестовые данные

Товары:
- Test.allTheThings() T-Shirt (Red)
- Sauce Labs Onesie
- Sauce Labs Fleece Jacket

## Шаги
1. Авторизоваться под валидным пользователем.
2. Добавить товар `Test.allTheThings() T-Shirt (Red)` в корзину.
3. Добавить товар `Sauce Labs Onesie` в корзину.
4. Добавить товар `Sauce Labs Fleece Jacket` в корзину.
5. Проверить счетчик корзины.

## Ожидаемый результат
- Все выбранные товары добавлены в корзину.
- Счетчик корзины отображает значение `3`.
- Цвет бейджа корзины соответствует ожидаемому.
