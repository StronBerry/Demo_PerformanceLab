# LOGIN_002 — Проверка ошибок авторизации

## Приоритет
Normal

## Функциональность
Авторизация

## Предусловия
- Приложение SauceDemo доступно.
- Открыта страница логина.

## Тестовые данные

| Сценарий | Логин | Пароль | Ожидаемое сообщение |
|---|---|---|---|
| Заблокированный пользователь | locked_out_user | secret_sauce | Epic sadface: Sorry, this user has been locked out. |
| Некорректный пользователь | incorrect_user | secret_sauce | Epic sadface: Username and password do not match any user in this service |
| Пустые данные | пусто | пусто | Epic sadface: Username is required |

## Шаги
1. Открыть страницу логина SauceDemo.
2. Ввести некорректные или пустые учетные данные.
3. Нажать кнопку Login.

## Ожидаемый результат
- Пользователь не авторизован.
- Отображается сообщение об ошибке.
- Текст ошибки соответствует ожидаемому значению.
