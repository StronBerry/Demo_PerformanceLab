# CART_002 — Проверка товаров в корзине

## Приоритет
Blocker

## Функциональность
Корзина

## Предусловия
- Приложение SauceDemo доступно.
- Пользователь авторизован.
- Открыта страница Products.

## Тестовые данные

Товары:
- Test.allTheThings() T-Shirt (Red)
- Sauce Labs Onesie
- Sauce Labs Fleece Jacket
- Sauce Labs Backpack

## Шаги
1. Авторизоваться под валидным пользователем.
2. Добавить все тестовые товары в корзину.
3. Проверить счетчик корзины.
4. Открыть корзину.
5. Проверить, что открыта страница корзины.
6. Проверить количество товаров в корзине.
7. Проверить наличие всех выбранных товаров в корзине.

## Ожидаемый результат
- Открыта страница корзины.
- В корзине отображаются все добавленные товары.
- Количество товаров соответствует ожидаемому значению `4`.
